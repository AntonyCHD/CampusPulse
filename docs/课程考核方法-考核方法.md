# 课程考核方法-考核方法

## 考核方法

- **分组：** 自行分组组建“创新创业小组”，每个组4~6人，每个小组自行推选1名组长。
- **选题：** 每个组围绕舆情、媒体以及相关联的人工智能与认知安全方向，选择1个为研究方向。
- **总成绩=平时成绩*50%+期末成绩*50%（百分制）**
  - **平时成绩包括：**
    - 小组创业答辩，满分40分。
    - 考勤，满分10分，点名未到未提前邮件请假（邮件内容：姓名、学号、请假时间、请假原因），每次扣2分。
  - **期末成绩**
    - 小组技术研究论文，满分50分。
    - 按照小组提交论文至少1篇。

## 课程主题结构

- **舆情与媒体安全**
  - **人工智能安全板块**
    - 人工智能与人工智能安全
    - 人工智能安全框架
    - 大模型安全对齐
    - 大模型安全测评
  - **舆情安全板块**
    - 舆情与舆情安全
    - 舆情大数据分析
    - 社群仿真与模拟
    - 大模型驱动舆情分析
  - **媒体安全板块**
    - 媒体与媒体安全
    - 人工智能生成内容
    - 内容安全审核
    - 深度伪造对抗
