## 🎯 核心算法

### 1. 评论链图谱构建
- **回复边**: parent_id关系
- **时间边**: 连续评论（30分钟内）
- **语义边**: 余弦相似度≥0.80

### 2. 风险评分公式
```
总分 = 0.20×主贴风险 + 0.25×集体共鸣 + 0.20×动员性
       + 0.15×图谱影响力 + 0.10×爆发性 + 0.10×不确定性
```

### 3. 风险等级划分
- 低: [0, 40)
- 中: [40, 65)
- 高: [65, 85)
- 严重: [85, 100]

## 📈 对比实验结果

### 方法对比
| 方法 | 优势 | 局限 |
|------|------|------|
| 关键词法 | 实现简单 | 无法理解语义，误判率高 |
| 情感分析法 | 考虑情感倾向 | 忽略传播结构，漏判集体行为 |
| 评论链演化法 | 语义+结构+传播三维分析 | 计算复杂度较高 |
