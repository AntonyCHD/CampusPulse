# 小组创新论文考核维度

## 要求

- 请深入理解技术研究与项目创业的区别，请勿直接复用《创业项目企划书》。
- 按照分组提交至少1篇论文，需要围绕“舆情与媒体安全”这一主题。
- 基本要求：符合论文规范、质量优先、结构完整、字数不限、请勿抄袭，请勿使用自动生成内容，中文论文。
- 可以使用SOTA工具加速研究过程。不必凑字数，请勿直接提交生成内容（文字、表格、图片、文献等等），如发现，每处扣分5分（百分制）。

## 作者计分系数

按照作者实际标注顺序计评分系数：

| 作者顺序 | 计分系数 |
| :--- | :--- |
| 第一作者 | 1.0 |
| 第二作者 | 0.95 |
| 第三作者 | 0.9 |
| 第四作者 | 0.85 |
| 第五作者及以后 | 0.8 |

同一作者参与多篇论文，该作者最终得分取参与论文个人得分平均分。
